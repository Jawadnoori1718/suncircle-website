# ☀️ SunCircle

**SunCircle is a peer-to-peer solar energy platform.** Buy cheaper local solar from a verified neighbour, or share and earn from your own panels — clean energy for the whole community.

This is the marketing and waitlist website for SunCircle.

---

## ✨ Features

- ⚡ Modern, responsive landing page built with React
- 📊 Interactive charts and data visualisations (Recharts)
- 📝 Waitlist sign-up connected to a Google Sheets backend
- 🔍 SEO-ready with Open Graph and Twitter meta tags
- 🎨 Clean, fast UI with Lucide icons

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| [React](https://react.dev/) | UI library |
| [Vite](https://vitejs.dev/) | Build tool & dev server |
| [Recharts](https://recharts.org/) | Charts |
| [Lucide React](https://lucide.dev/) | Icons |
| Google Apps Script | Waitlist backend |

## 🚀 Getting Started

Make sure you have [Node.js](https://nodejs.org/) installed, then:

```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev
```

Then open the local URL shown in your terminal (usually `http://localhost:5173`).

### Other commands

```bash
npm run build     # Build for production (output goes to /dist)
npm run preview   # Preview the production build locally
```
